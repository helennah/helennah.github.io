BIOL 101

Unit 2 Project

This project uses the Palmer Penguins dataset to explore data visualization techniques in R using ggplot2. I created bar charts, histograms, boxplots, and scatterplots to compare body mass, flipper length, and bill dimensions across three penguin species and two sexes. The project also covers data cleaning using regular expressions to fix messy species, sex, and island data.

Unit 3 Project

This project analyzes morphological data from Edgar Anderson's classic Iris dataset using R, including data visualization and Linear Discriminant Analysis (LDA) to classify three Iris species. It also maps geographical occurrence data pulled from the GBIF database to explore where each species is found in the wild.
